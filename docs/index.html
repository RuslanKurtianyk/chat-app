Повідомлення (HTTP)

Прочитано: readBy у кожному повідомленні. Позначити:

Зображення (спільне завантаження)

POST /storage/upload — зображення в Cloudinary; поле форми file, опційно kind: general | profile | message | story. Повертає { url, mimeType, … }; далі підстав URL у профіль або сторіз.

Файл у чат (HTTP → бекенд → Cloudinary)

Клієнт надсилає файл на POST /messages/upload; сервер завантажує в Cloudinary і записує attachmentUrl у повідомлення. Потрібні CLOUDINARY_CLOUD_NAME, CLOUDINARY_API_KEY, CLOUDINARY_API_SECRET на сервері.

Гаманець / валюта (HTTP)

Баланс і транзакції в minor units (цілі числа). Валюта за замовчуванням: COIN.
